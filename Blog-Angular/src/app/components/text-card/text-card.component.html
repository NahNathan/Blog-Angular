Sobre mim

Olá! Sou apaixonado por tecnologia e desenvolvimento de software. Me formei em Análise e Desenvolvimento de Sistemas pelo IFTM e hoje trabalho como desenvolvedor full-stack na Codesquare. Gosto de estar sempre aprendendo algo novo, seja uma tecnologia, framework ou até mesmo uma abordagem diferente para resolver problemas. Acredito que um bom código vai além de funcionar: ele deve ser limpo, fácil de entender e contribuir para que o projeto cresça de forma saudável. Curto trocar ideias sobre tecnologia, experimentar soluções e, principalmente, ver o resultado do trabalho impactando positivamente as pessoas.
